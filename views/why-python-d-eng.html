PYTHON

DATA ENGINEER

Python se usa ampliamente en Ingeniería de datos porque es fácil de aprender y leer. Y tiene la capacidad de interactuar con algoritmos escritos en lenguajes como C. Aparte de usarse en la limpieza de datos, se usa en Inteligencia Artificial, análisis predictivo y Machine Learning.

  • Apache Airflow: Es una plataforma que permite crear, programar y monitorear flujos de trabajo mediante programación.
  • Pyspark: Es la API de Python para Apache Spark, un marco informático distribuido de código abierto y un conjunto de bibliotecas para el procesamiento de datos a gran escala en tiempo real.
  • Las notebooks como Jupyter Notebook y Google Colab proporcionan herramientas convenientes para la visualización de datos y la investigación exploratoria.
