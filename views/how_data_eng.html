Modelos Cloud

  • Nube Pública: No requiere gastos de capital para escalar. Las organizaciones pagan sólo por lo que usan. Las aplicaciones se pueden aprovisionar y desaprovisionar rápidamente.
  • Nube Privada: Las organizaciones tienen control completo sobre los recursos y la seguridad.
  • Nube Híbrida: Tiene mayor flexibilidad. Las organizaciones determinan dónde correr sus aplicaciones, controlan la seguridad, cumplimiento de políticas y requerimientos legales.

ETL

ETL son las siglas de Extract, Transform and Load. Es el proceso en el que los datos se extraen de cualquier fuente de datos y se transforman en un formato adecuado para almacenarlos y con fines de referencia futuros.

  • SSIS
  • Pentaho Data Integration
  • Talend
  • Apache NiFi
  • SAS
  • Informatica Power Center
  • Azure Data Factory
  • AWS Glue
  • QlikView

PYTHON

Python se usa ampliamente en Ingeniería de datos porque es fácil de aprender y leer. Y tiene la capacidad de interactuar con algoritmos escritos en lenguajes como C. Aparte de usarse en la limpieza de datos, se usa en Inteligencia Artificial, análisis predictivo y Machine Learning.

  • Apache Airflow: Es una plataforma que permite crear, programar y monitorear flujos de trabajo mediante programación.
  • Pyspark: Es la API de Python para Apache Spark, un marco informático distribuido de código abierto y un conjunto de bibliotecas para el procesamiento de datos a gran escala en tiempo real.
  • Las notebooks como Jupyter Notebook y Google Colab proporcionan herramientas convenientes para la visualización de datos y la investigación exploratoria.
